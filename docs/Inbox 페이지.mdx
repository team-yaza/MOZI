# Inbox 페이지

여기에 동영상 삽입 (Inbox에서 할 수 있는 전체적인 일)

Inbox 페이지에서 할 수 있는 전체적인 기능입니다. 모든 기능은 오프라인 동기화 기능이 적용되어 있어서 오프라인에서 수정해도 인터넷이 연결되는 순간 서버에 반영됩니다.

## 할 일 삭제

Inbox 페이지에서는 할 일을 생성, 수정, 삭제 할 수 있습니다. 할 일은 기본적으로 Title과 Description으로 구성되어있습니다. 각 할 일을 더블 클릭하면 할 일이 열리고, 세부적인 사항을 등록할 수 있습니다.

할 일을 어디에서 할지, 언제까지 할지, 알림 설정을 할 수 있고, 할 일을 완료하면 체크박스를 클릭해면 완료 처리를 할 수 있습니다.

이 모든 과정은 오프라인에서도 작동하기 때문에 오프라인에서 할 일을 작성하고 동기화 기능을 통화여 인터넷이 연결되면 자동으로 서버에 업로드 됩니다.
